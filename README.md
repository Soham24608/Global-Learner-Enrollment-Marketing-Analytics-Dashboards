# 📊 Global Learner Enrollment & Marketing Analytics Dashboards

## 📌 Project Overview

This project presents **interactive data visualization dashboards** built using **Google Looker Studio** to analyze global learner enrollment trends and marketing campaign performance.

The dashboards transform raw platform data into **actionable insights**, helping understand learner demographics, geographic participation, opportunity popularity, and marketing effectiveness.
